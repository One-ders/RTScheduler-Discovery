## Hardware Interfaces

### SPI Interface

- **Pins Used:** GPIO10, GPIO11, GPIO12, GPIO13
- **Description:** SPI communication with external devices.

### I2C Interface

- **Pins Used:** GPIO20, GPIO21
- **Description:** I2C communication with sensors.

### UART Interface

- **Pins Used:** GPIO16, GPIO17
- **Description:** UART communication for serial communication.

### PWM Outputs

- **Pins Used:** GPIO15, GPIO14
- **Description:** Pulse Width Modulation outputs for motor control and LED dimming.

### ADC Inputs

- **Pins Used:** GPIO32-39
- **Description:** Analog to Digital Converter inputs for sensors.
